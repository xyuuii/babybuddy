# Code of Conduct

## Our Pledge

We want babybuddy to be a welcoming project for people building thoughtful family software. Contributors should be respectful, patient, and privacy-conscious.

## Expected Behavior

- Be kind and constructive.
- Assume good intent while still asking for clarity.
- Respect that family data, photos, and baby records are sensitive.
- Avoid sharing private media, credentials, API keys, or personal server details in public discussions.

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory language.
- Publishing another person's private information.
- Posting real credentials, private NAS URLs, family media, or sensitive medical details.
- Pressuring maintainers or contributors to disclose private data for debugging.

## Enforcement

Maintainers may edit, hide, or remove comments and issues that violate this code of conduct. Repeated or severe violations may result in being blocked from participating in the project.
